Assignment Questions and Answers

  1. What is the purpose of the `id` attribute?

    The id attribute provides a unique identifier for an HTML element, which can be used in CSS and JavaScript.

  2. How can the `class` attribute be useful?

    The class attribute allows multiple elements to share the same styling and can be used for JavaScript manipulation.

  3. What does the `style` attribute do?

    The style attribute applies inline CSS styles directly to an HTML element.

  4. How does the `title` attribute enhance usability?

    The title attribute provides additional information via tooltips when hovering over elements.

  5. What are `data-*` attributes used for?

    Custom data-* attributes store extra information about an element that can be accessed via JavaScript.

  6. How does `tabindex` affect navigation?

    The tabindex attribute specifies the tab order for keyboard navigation.

  7. What does the `lang` attribute specify?

    The lang attribute specifies the language of the element's content.

  8. What is the function of the `hidden` attribute?

    The hidden attribute indicates that the element is not relevant and should not be displayed.

  9. Can an element have multiple classes?

    Yes, an element can have multiple classes by separating them with spaces in the class attribute.

  10. Is it a good practice to use inline styles?

    While inline styles can be convenient, it's generally better to use external stylesheets for maintainability and consistency.
