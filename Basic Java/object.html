Array Methods Tasks

5. Find the index of `15` in the `numbers` array.

7. Remove the third element from `["Red", "Green", "Blue", "Yellow"]`.
8. Convert `["JavaScript", "is", "awesome"]` into `"JavaScript is awesome"`.
9. Reverse `[9, 8, 7, 6, 5]`.
10. Sort `[9, 3, 7, 1, 5]` in ascending order.
11. Merge `[1, 2, 3]` and `[4, 5, 6]`.
12. Check if all elements in `[2, 4, 6, 8]` are even numbers.
13. Check if any number in `[10, 20, 30, 5]` is greater than `25`.
14. Replace middle elements with `0` in `[5, 10, 15, 20]`.
15. Copy elements within `[1, 2, 3, 4, 5]`.
