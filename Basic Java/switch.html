Switch Case Tasks

Simple Calculator

Create a basic calculator that can perform addition, subtraction, multiplication, and division. Users will input two numbers and an operation symbol (+, -, *, /) to get the result.

Day of the Week

Ask the user to enter a number between 1 and 7. Display the corresponding day of the week (e.g., 1 for Sunday, 2 for Monday, etc.).

Grade System

Based on a student's score (0-100), display their grade: 'A' for 90 and above, 'B' for 80-89, 'C' for 70-79, 'D' for 60-69, and 'F' for below 60.

Traffic Light Simulator

Simulate a traffic light system where users input a color ('red', 'yellow', 'green') and display the appropriate action ('STOP', 'READY', 'GO').

Currency Converter

Allow users to input an amount in USD and convert it to EUR, GBP, or INR based on the chosen currency.
