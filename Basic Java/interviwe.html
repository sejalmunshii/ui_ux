Section 1: JavaScript Introduction (10 Questions)

Q1. JavaScript kya hai?

Ans: JavaScript ek high-level, interpreted programming language hai jo browsers mein chalti hai. Yeh web pages ko dynamic aur interactive banata hai.

Explanation: HTML structure provide karta hai, CSS styling, aur JavaScript functionality.

Q2. JavaScript client-side ya server-side language hai?

Ans: JavaScript primarily client-side hai, but Node.js ke through yeh server-side bhi chal sakta hai.

Explanation: Node.js JavaScript ka server-side runtime hai.

Q3. JavaScript ko HTML file mein kaise add karte hain?

Ans:

<script src="script.js"></script> <!-- ya --> <script> // JS code </script>
Q4. JavaScript aur Java mein kya difference hai?

Ans: Java ek statically typed, compiled language hai, jabki JavaScript dynamically typed aur interpreted language hai.

Q5. JavaScript ka engine kya hota hai?

Ans: JavaScript engine browser mein hota hai, jaise Chrome mein V8 engine.

Q6. JavaScript synchronous ya asynchronous hai?

Ans: JavaScript synchronous hai, lekin asynchronous operations support karta hai (callbacks, promises, async/await se).

Q7. Browser JavaScript ko kaise run karta hai?

Ans: Browser ka JS engine code ko parse karta hai, aur memory allocate karke execute karta hai.

Q8. JavaScript mein loosely typed ka kya matlab hai?

Ans: JavaScript mein variables ka type define nahi karna padta, runtime par automatically decide hota hai.

Q9. JavaScript ka use sirf browser mein hota hai kya?

Ans: Nahi, Node.js ke through JavaScript backend mein bhi use hota hai.

Q10. JavaScript code ko debug kaise karte hain?

Ans: Browser ke Developer Tools (Inspect > Console) ka use karke hum debug kar sakte hain.

üßÆ Section 2: Variables and Data Types (15 Questions)

Q11. JavaScript mein variables kaise declare karte hain?
var x = 10; let y = 20; const z = 30;
Q12. var, let aur const mein kya difference hai?

var: function-scoped, redeclare allowed

let: block-scoped, redeclare not allowed

const: block-scoped, value change nahi kar sakte

Q13. JavaScript mein primitive data types kaunse hain?

Ans: String, Number, Boolean, Null, Undefined, Symbol, BigInt

Q14. typeof operator kya karta hai?

Ans: Variable ka type batata hai.
Example: typeof "hello" returns "string"

Q15. Undefined aur null mein kya difference hai?

undefined: variable declare hua hai par value assign nahi hui.

null: intentionally khali value assign ki gayi hai.

Q16. NaN ka matlab kya hai?

Ans: Not-a-Number, yeh tab aata hai jab numeric calculation invalid ho.
Example: parseInt("abc") // NaN

Q17. JavaScript mein dynamic typing kya hoti hai?

Ans: Variable ka type runtime par change ho sakta hai.

let a = 10; a = "hello"; // valid
Q18. const object ki value change kar sakte hain kya?

Ans: Object ka reference change nahi kar sakte, par uske properties change kar sakte hain.

Q19. BigInt kya hai?

Ans: Bahut bade numbers ko store karne ke liye use hota hai.
Example: let big = 12345678901234567890n;

Q20. Boolean type ka example do.
let isLogin = true; let hasAccess = false;

üî§ Section 2: Strings & Hoisting (Q21‚ÄìQ25)

Q21. JavaScript mein string declare kaise karte hain?
Ans: Single ('hello'), double ("hello") ya template literals (`hello`) se.
Q22. Template literals kya hain?
Ans: Backticks (`) ka use karke string interpolation kar sakte hain. 
let name = "Raj";
  console.log(`Hello, ${name}`);
Q23. typeof null kya return karta hai?
Ans: "object" (yeh ek historical bug hai)
Q24. typeof NaN kya return karta hai?
Ans: "number"
Q25. var hoisting kya hai?
Ans: JavaScript mein var declarations top par move ho jati hain execution se pehle.

üõ†Ô∏è Section 3: Methods (Q26‚ÄìQ35)

Q26. Method kya hota hai?
Ans: Function jo kisi object ke andar defined hota hai. 
let person = {
    greet: function() {
      return "Hello";
    }
  };
Q27. Function declaration aur expression mein kya farak hai?
Ans:
  • Declaration: function greet() {}
  • Expression: let greet = function() {};
Q28. Arrow function kya hota hai?
Ans: Short syntax hoti hai. 
const greet = () => "Hello";
Q29. Arrow function aur normal function mein difference?
Ans: Arrow function this bind nahi karta, aur implicit return allow karta hai.
Q30. String method example do.
let str = "hello";
  console.log(str.toUpperCase()); // "HELLO"
Q31. Number method example?
let num = 123.456;
  console.log(num.toFixed(2)); // "123.46"
Q32. parseInt() aur parseFloat() mein difference?
Ans: parseInt() integer return karta hai, parseFloat() decimal allow karta hai.
Q33. Object method call kaise karte hain?
obj.methodName();
Q34. isNaN() function kya karta hai?
Ans: Check karta hai ki value NaN hai ya nahi. 
isNaN("abc") // true
Q35. Array ka length method kya karta hai?
Ans: Array ke elements ki total count return karta hai.

üîÅ Section 4: Conditional Statements (Q36‚ÄìQ50)

Q36. Conditional statement kya hoti hai?
Ans: Logic ke base par code run karna (if-else, switch)
Q37. if-else statement ka syntax kya hota hai?
if (condition) {
    // code
  } else {
    // code
  }
Q38. Ternary operator kya hota hai?
Ans: Short form of if-else 
let result = age >= 18 ? "Adult" : "Minor";
Q39. == aur === mein kya difference hai?
Ans:
  • == type conversion karta hai
  • === strict equality check karta hai (type bhi match hona chahiye)
Q40. switch statement ka use kab karte hain?
Ans: Jab multiple conditions check karni ho same variable ke against.
Q41. Nested if-else kya hota hai?
Ans: if ke andar dusra if statement. 
if (a > 0) {
    if (a < 10) {
      console.log("0 to 10");
    }
  }
Q42. JavaScript mein falsy values kaun si hain?
Ans: false, 0, "", null, undefined, NaN
Q43. if condition mein string check kar sakte hain kya?
Ans: Haan, empty string false hoti hai, non-empty true.
Q44. JavaScript mein short-circuiting kya hoti hai?
Ans: || aur && ka use karke default value assign karna. 
let name = userInput || "Guest";
Q45. if (0) chalega kya?
Ans: Nahi, kyunki 0 falsy hota hai.
Q46. if ('0') chalega kya?
Ans: Haan, kyunki '0' ek non-empty string hai (truthy).
Q47. Multiple conditions kaise likhte hain?
if (a > 0 && a < 100) { ... }
Q48. else if statement kya hota hai?
Ans: Multiple conditions chain karne ke liye use hota hai.
Q49. Conditional statements mein scope ka kya role hai?
Ans: let/const block scope mein confined rehte hain.
Q50. Truthy aur falsy ka use kab hota hai?
Ans: Jab koi variable directly condition mein use karte hain.
