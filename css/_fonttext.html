CSS Font and Text Styling Interview Task

Theory Questions

  1. Explain the following CSS font properties and their purposes:
    • font-family
    • font-size
    • font-weight
    • font-style
    • font-variant
    • line-height
    • letter-spacing
    • text-transform
    • text-align
    • text-decoration
  2. What is the difference between em, rem, px, %, and vw units in CSS when setting font sizes?
  3. What are web-safe fonts, and why are they important?
  4. Explain the concept of font fallbacks in CSS. Why is it important to provide fallback fonts?
  5. How does the @font-face rule work? What are the advantages and disadvantages of using custom fonts in CSS?
  6. What is the purpose of font-display when using @font-face, and what are the different options available?
  7. Describe how to center text horizontally and vertically inside a div. What challenges could arise from different scenarios?
  8. What is text rendering, and how does it affect performance? Discuss how text-rendering can be used in CSS.

Practical Task

Create a webpage that demonstrates different font and text properties using CSS as per the following requirements:

HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Font and Text Styling</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="header">
    <h1>Font and Text Styling Task</h1>
  </div>
  
  <div class="content">
    <p class="serif-text">This is a paragraph using a serif font.</p>
    <p class="sans-serif-text">This is a paragraph using a sans-serif font.</p>
    <p class="italic-text">This text should be italicized.</p>
    <p class="bold-text">This text should be bold.</p>
    <p class="uppercase-text">This text should be in uppercase.</p>
    <p class="spaced-text">This text should have increased letter spacing.</p>
    <p class="centered-text">This text should be centered both horizontally and vertically.</p>
  </div>
  
  <div class="footer">
    <p>End of the font and text styling task.</p>
  </div>
</body>
</html>

End of the font and text styling task.
